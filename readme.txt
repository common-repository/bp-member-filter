== Description ==

Allows filtering of member data in BuddyPress members directory by extended profile field.

(NOTE: Still does not work with checkboxes.)

== Installation ==

1. Upload `bp-member-filter.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add a form to your members/index.php or some other file (See: http://pastebin.com/bw0suXwe)
1. Smile

== Frequently Asked Questions ==

= Do I need to edit my theme? =

Yes. You will need to figure out where you want to put the form that will contain the searchable fields. This will probably change in 1.1, but for now you've got functions to help you.
